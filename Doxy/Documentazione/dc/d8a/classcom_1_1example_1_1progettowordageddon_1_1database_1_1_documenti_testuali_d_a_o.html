Riferimenti per la classe DocumentiTestualiDAO
Database

Descrizione

Classe per la gestione dei documenti testuali nel database.

Fornisce metodi per il recupero, l'inserimento, la modifica e l'eliminazione di documenti dalla tabella DocumentiTestuali, nonché per la ricerca tramite indice sulla lingua.

Struttura della tabella DocumentiTestuali nel database:

Campo Tipo Vincoli Descrizione
nome TEXT PRIMARY KEY Nome univoco del documento
lingua TEXT NOT NULL CHECK IN ('ITALIANO', 'INGLESE', 'FRANCESE', 'SPAGNOLO') Lingua del documento
difficolta TEXT NOT NULL CHECK IN ('FACILE', 'MEDIA', 'DIFFICILE') Livello di difficoltà del documento
testo CLOB NOT NULL Contenuto testuale del documento

Indici creati:

  • idx_doc_lingua su campo lingua per velocizzare le query per lingua
