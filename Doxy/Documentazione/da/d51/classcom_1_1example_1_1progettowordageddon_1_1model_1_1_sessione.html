Riferimenti per la classe Sessione
Model

Descrizione

Classe che rappresenta lo stato globale della sessione utente nell'applicazione.

Questa classe tiene traccia delle informazioni relative alla sessione corrente, come l'utente attivo, lo stato del quiz, la schermata corrente e la difficoltà selezionata.

Inoltre, implementa metodi per salvare automaticamente la sessione in un file .wordageddon nella cartella Sessioni del progetto, e importare i dati da quest'ultimo per ripristinare sessioni chiuse precedentemente.

Metodo di utility per aggiornare il nome del file di sessione.

Ricostruisce il nome del file in base alla schermata attuale.

Nota
PannelloControllo-09_giu_25-20_40-15041.wordageddon è presumibilmente il nome di una sessione creata il 09 giugno 2025 alle 20:40, dove l'ultima schermata aperta è stata "PannelloControllo".

◆ caricaSessione()

void caricaSessione ( String pathFile) throws IOException, ClassNotFoundException, IllegalArgumentException

Carica una sessione da file .wordageddon.

Aggiorna i valori della sessione corrente in base a quelli contenuti nel file.

Parametri
pathFilePercorso assoluto del file da cui caricare la sessione.
Eccezioni
IOExceptionse il file è danneggiato o non corrisponde all'attuale versione dell'applicazione.
ClassNotFoundExceptionse una delle classi della sessione salvata non è più presente nell'applicazione.
IllegalArgumentExceptionse l'utente della sessione caricata non corrisponde a quello attualmente loggato.
