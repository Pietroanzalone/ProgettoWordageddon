Descrizione

Gestisce la stampa dei messaggi di log per sviluppatori.

La classe implementa un sistema di logging user-friendly che mostra a terminale (System.out) messaggi come informazioni o errori dell'applicazione.

◆ loggingDisattivato()

static boolean loggingDisattivato ( )
staticprivate

Verifica se il logging è disattivato per la sessione corrente.

Restituisce
true se il logging è disattivato (sessione nulla o logging non attivo), false se il logging è attivo.
Nota
Questo metodo viene chiamato prima di ogni operazione di logging per evitare output non desiderato quando il logging è disabilitato.
