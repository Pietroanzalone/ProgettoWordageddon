Riferimenti per la classe UtentiDAO
Database

Descrizione

Classe per la gestione degli utenti nel database.

Fornisce metodi per il recupero, l'inserimento, la modifica e l'eliminazione di utenti dalla tabella Utenti, nonché per il controllo dell'esistenza di un utente.

Struttura della tabella Utenti nel database:

Campo Tipo Vincoli Descrizione
username TEXT PRIMARY KEY Identificatore univoco dell'utente
password TEXT NOT NULL Password dell'utente (potenzialmente hashata)
admin BOOLEAN NOT NULL Indica se l'utente ha privilegi amministrativi
