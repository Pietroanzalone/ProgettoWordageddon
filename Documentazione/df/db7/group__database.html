Descrizione

Contiene tutte le classi e componenti necessarie alla comunicazione con il database SQLite.

Questo modulo include tutte le classi relative alla creazione, aggiunta e rimozione di tutti i dati, ovvero utenti, domande, documenti e sessioni di gioco.

Database Access Object

La classe DAO fornisce i metodi di utility per eseguire le query SELECT, INSERT e DELETE. Ogni classe DAO gestisce una tabella del database e fornisce metodi per accedervi o modificarla.
