UI

Descrizione

Il controller principale per la gestione della UI.

Implementa funzioni di base, come il cambio di schermata e la chiusura dell'applicazione. Inoltre, gestisce interamente la logica della barra di navigazione in alto.

◆ initialize()

void initialize ( )

Metodo chiamato automaticamente all'inizializzazione del controller.

Inizializza la barra di navigazione usando impostaNavbar, impostaIconaLogout e impostaIconaHome.

Nota
Se un controller ha la barra di navigazione che non comprende tutti e tre i pulsanti, deve assicurarsi di fare l'Override di questo metodo e usare solo i comandi che gli servono.
Esempio per AccediController che ha solo i pulsanti "Esci" e "Logout":
@Override
public void initialize() {
impostaNavbar();
impostaIconaLogout();
}

◆ caricaImmagine()

ImageView caricaImmagine ( String nome,
int larghezza,
int altezza )
protected

Metodo di utility per caricare un'immagine dalla cartella Assets del progetto e la restituisce come oggetto ImageView.

L'immagine viene scalata alle dimensioni specificate tramite i parametri larghezza e altezza.

Parametri
nomeIl nome del file immagine, inclusa l'estensione (es. "logo.png").
larghezzaLa larghezza desiderata per l'icona in pixel.
altezzaL'altezza desiderata per l'icona in pixel.
Restituisce
Un oggetto ImageView contenente l'immagine scalata, oppure null se il file non viene trovato.
