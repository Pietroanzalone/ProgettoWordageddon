Database

Descrizione

Classe per l'accesso al database SQLite.

Questa classe fornisce metodi protetti statici per eseguire query di selezione e aggiornamento sul database tramite JDBC.

Esegue una query SELECT parametrizzata e restituisce i risultati.

Abilita le foreign key, prepara la query, imposta i parametri e restituisce la lista di risultati come array di oggetti.
